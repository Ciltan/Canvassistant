# Row Space, Column Space, Nullspace, and the Rank-Nullity Theorem

> [!info] Navigation
> ↑ Back to [[MOD_INDEX]]

## Summary
In this lecture, we explore the concepts of row space, column space, and nullspace, which are fundamental in understanding the structure of matrices. We delve into the definition of nullity and its relationship with the nullspace and general solutions of linear equations. The rank-nullity theorem is introduced, illustrating the relationship between the rank and nullity of a matrix. Through exercises, we apply these concepts to practical scenarios, reinforcing the theoretical insights into real-world applications.

## Key Concepts
- [[Row Space]] — equivalent via row operations.
- [[Column Space]] — pivot columns are equivalent via row operations.
- [[Nullspace]] — the solution space to the homogeneous equation.
- [[Nullity]] — the dimension of the nullspace or parameters for the general solution of the homogeneous equation.
- [[Rank-Nullity Theorem]] — $ \text{rank}(A) + \text{nullity}(A) = n $, where $n$ is the number of columns of $A$.

## Detailed Notes

### Row Space, Column Space, and Nullspace
- **Row Space:** The set of all possible linear combinations of the row vectors of a matrix. Equivalent via row operations.
- **Column Space:** The set of all possible linear combinations of the column vectors. Pivot columns from reduced row echelon form (RREF) are representatives.
- **Nullspace:** The set of all solutions $\mathbf{x}$ to the homogeneous equation $A\mathbf{x} = \mathbf{0}$.

### Nullity
- Defined as the dimension of the nullspace.
- Represents the number of free parameters in the general solution to the homogeneous system $A\mathbf{x} = \mathbf{0}$.

### Rank-Nullity Theorem
Given a matrix $A$ with $n$ columns:
$$ \text{rank}(A) + \text{nullity}(A) = n $$
